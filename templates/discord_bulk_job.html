JSON File Format Example

Your JSON file should contain an array of objects with the following format:

Example JSON: 
[
  {
    "name": "Job 1",
    "message_link": "https://discord.com/channels/123456789/987654321/111222333"
  },
  {
    "name": "Job 2", 
    "message_link": "https://discord.com/channels/123456789/987654321/444555666"
  }
]

Each object should have a "name" and "message_link" field.

Requirements:

  • File must be valid JSON format
  • Each object must have "name" and "message_link" properties
  • Values cannot be empty
  • Message links must be valid Discord message URLs
  • Each Discord message must have exactly 8 attachments (4 audio + 4 images)
  • Posts will be sent to n8n webhooks at specified intervals
